Проект
Измерение температуры и влажности воздуха при помощи датчика dht11
Описание кода:
В кода настраиваем работу RTC таймера и с помощью него совершаем прерывания каждые 2 секунды, в котором проводим измерение температуры и влажности и вывод их на 
семисегментные индикатор.
Общение с датчиком осуществляем по протоколу, для измерения времени сигналов от датчика используем встроенный таймер.
Считываем данные с датчика в функции  int dht11_get_inform(), затем обрабатываем их переводя биты в значения температуры и влажности в функции void decoding().
В теле main выводим значения температуры или влажности при помощи функции void dynamic_indication(), 
по нажатию кнопки меняется выводимое значение (Температура или влажность)
Для работы с семисегментым индикатаром используем функцию static void set_indicator(uint8_t number)
